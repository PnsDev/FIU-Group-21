Books

Up 
get /books/create
Create a new book (createBook)

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/json

Return type

BasicResponse

Example data

Content-Type: application/json
{
  "success" : true,
  "message" : "message"
}

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

Book was successfuly saved BasicResponse

Example data

Content-Type: response
{"successful":true,"message":"Book saved"}

400

Body is not valid or Book already exists BasicResponse

Example data

Content-Type: response
{"successful":false,"message":"Invalid book provided"}

401

Invalid authorization header BasicResponse

Example data

Content-Type: response
{"successful":false,"message":"Invalid authorization header"}

500

Internal server error BasicResponse

Example data

Content-Type: response
{"successful":false,"message":"Internal server error"}
